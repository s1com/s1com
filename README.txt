СЕРВИС.COM — САЙТ ЦЕЛИКОМ В НОВОМ ДИЗАЙНЕ (+ страница товара)
============================================================
Один деплой на ВЕБ-сервис. Импорт не нужен. Заливать всё разом.

НОВОЕ В ЭТОЙ СБОРКЕ:
  • Страница товара переведена на новый дизайн (SSR — хорошо для SEO):
    галерея с миниатюрами слева, справа — бренд, H1, артикул, цена/наличие,
    кнопки «+ В заявку», WhatsApp, «Запросить оптовую цену», преимущества;
    ниже вкладки (Характеристики / Описание / Доставка) и блок «Похожие
    товары». Тот же header + мегаменю + корзина, что и на разделах.
  • Корзина общая на всём сайте (один ключ): добавил на разделе — видно в
    заявке на странице товара и наоборот.

ВЕСЬ САЙТ В НОВОМ ВИДЕ:
  главная + 6 разделов (видео, сетевое, скуд, пожарная, ИБП, кабельные) +
  страница товара. Единый header, мегаменю, карточки, корзина.

ФАЙЛЫ:
  server.js                       — API (home, поиск, group), маршрут товара.
  lib/product-page.js             — НОВЫЙ дизайн страницы товара (SSR).
  public/js/product.js            — НОВЫЙ. Логика страницы товара.
  public/js/home.js, catalog.js   — главная и разделы.
  public/css/catalog.css          — общие стили.
  public/*.html                   — главная + 6 разделов.

ДЕПЛОЙ:
 1) GitHub → Upload files → server.js (в корень) + папки public/ и lib/
    целиком (ВКЛЮЧАЯ public/js/product.js и lib/product-page.js) → Commit.
 2) Render → веб-сервис → Manual Deploy → Clear build cache & deploy.
 3) Cmd+Shift+R.

ПРОВЕРЬ:
 [ ] Открой любой товар (клик по карточке в разделе) → новый дизайн:
     галерея, вкладки, похожие, «+ В заявку».
 [ ] «+ В заявку» на товаре → панель справа; счётчик в шапке.
 [ ] Перейди в раздел — товар остаётся в заявке (общая корзина).
 [ ] Отправь заявку → в админке видна услуга (раздел) и страница-источник.
 [ ] Вкладки переключаются; миниатюры меняют главное фото.

ОСТАЁТСЯ (по желанию): страница брендов, избранное/сравнение (отдельная
страница), SSR-товары на разделах для SEO, подключение домена servis-com.kz.
