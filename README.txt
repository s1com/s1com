СЕРВИС.COM — НОВЫЙ КАТАЛОГ ЦЕЛИКОМ (главная + все разделы)
==========================================================
Один деплой на ВЕБ-сервис. Импорт не нужен. Заливать всё разом.

ЧТО ВНУТРИ:
  • Главная — новый лендинг (мегаменю, поиск, сегменты, категории, хиты,
    новинки, бренды, доверие).
  • ВСЕ 6 разделов в новом дизайне: Видеонаблюдение, Сетевое, СКУД,
    Пожарная + НОВЫЕ ИБП и Кабельные. На каждом: новый header + мегаменю,
    хлебные крошки, H1, подкатегории, фильтры-фасеты (бренд с поиском +
    тип + разрешение + цена + наличие, счётчики, мультивыбор, сброс,
    на мобильном — «Фильтры»), сортировка, новые карточки, «Показать ещё»,
    корзина-заявка (в базу через /api/order).

ФАЙЛЫ:
  server.js                 — /api/home, поиск ?q=, фильтр ?group=.
  public/index.html         — главная (футер обновлён под 6 разделов).
  public/js/home.js         — логика главной + мегаменю.
  public/js/catalog.js      — НОВЫЙ. Логика страницы категории.
  public/css/catalog.css    — НОВЫЙ. Стили страницы категории.
  public/videonablyudenie.html, setevoe.html, skud.html,
  public/pozharnaya.html, ibp.html (новый), kabelnye.html (новый)

ДЕПЛОЙ:
 1) GitHub → Upload files → server.js (в корень) + папку public/ целиком
    (ВКЛЮЧАЯ новые public/js/catalog.js и public/css/catalog.css) → Commit.
 2) Render → веб-сервис → Manual Deploy → Clear build cache & deploy.
 3) Открой сайт с Cmd+Shift+R.

ПРОВЕРЬ:

     плитки всех 6 разделов.
