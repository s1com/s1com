СЕРВИС.COM — ПОЛНЫЙ САЙТ + СТРАНИЦА БРЕНДОВ (актуальная сборка)
==============================================================
Один деплой на ВЕБ-сервис. Импорт не нужен. Самая свежая версия —
заливать поверх любой предыдущей.

НОВОЕ В ЭТОЙ СБОРКЕ:
  • Страница брендов /brands.html:
    – сетка всех брендов со счётчиком товаров;
    – клик по бренду → витрина бренда: описание (для известных брендов),
      фильтр по разделу, карточки товаров, корзина;
    – пункт «Бренды» в меню теперь ведёт сюда (на всех страницах);
    – плитки брендов на главной кликабельны → страница бренда.

ВЕСЬ САЙТ В НОВОМ ДИЗАЙНЕ:
  главная (лендинг) · 6 разделов (фасеты/карточки/пагинация) ·
  страница товара (галерея/вкладки/похожие) · страница брендов ·
  единый header/мегаменю/корзина · SSR-товары на разделах для SEO.

  server.js                 — API (+ /api/brands, ?brand=) + SSR + товар.

  public/js/home.js         — главная.
  public/js/catalog.js      — разделы.
  public/js/product.js      — страница товара.
  public/js/brands.js       — НОВЫЙ. Страница брендов.
  public/css/catalog.css    — общие стили.
  public/index.html, videonablyudenie.html, setevoe.html, skud.html,
  public/pozharnaya.html, ibp.html, kabelnye.html, brands.html (новый)

ДЕПЛОЙ:
 1) GitHub → Upload files → server.js (в корень) + папки public/ и lib/
