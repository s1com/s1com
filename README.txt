СЕРВИС.COM — ПОЛНЫЙ САЙТ (актуальная сборка, deploy этим архивом)
=================================================================
Один деплой на ВЕБ-сервис. Импорт не нужен. Это самая свежая версия —
можно заливать поверх любой предыдущей.

ВЕСЬ САЙТ В НОВОМ ДИЗАЙНЕ:
  • Главная — лендинг (мегаменю, поиск, сегменты, категории, хиты, бренды).
  • 6 разделов — фасеты со счётчиками, карточки, пагинация, корзина.
  • Страница товара — галерея, вкладки, похожие, «В заявку».
  • Единый header/мегаменю/корзина на всём сайте.

НОВОЕ В ЭТОЙ СБОРКЕ:
  • SSR-товары на страницах разделов: список товаров теперь есть прямо в
    HTML (важно для индексации, особенно Яндексом — ключевой поиск в РК).
    JS при загрузке заменяет его интерактивной сеткой, дублей нет.
  • SSR включён и для новых разделов ИБП и «Кабельные».

ФАЙЛЫ:
  server.js                 — API + SSR товаров разделов + маршрут товара.
  lib/product-page.js       — страница товара (новый дизайн, SSR).
  public/js/product.js      — логика страницы товара.
  public/js/home.js         — логика главной.
  public/js/catalog.js      — логика разделов (фасеты/карточки/корзина).
  public/css/catalog.css    — общие стили каталога.
  public/index.html         — главная.
  public/videonablyudenie.html, setevoe.html, skud.html,
  public/pozharnaya.html, ibp.html, kabelnye.html

ДЕПЛОЙ:
 1) GitHub → Upload files → server.js (в корень) + папки public/ и lib/
    целиком → Commit.
 2) Render → веб-сервис → Manual Deploy → Clear build cache & deploy.
 3) Cmd+Shift+R.

ПРОВЕРКА SEO (SSR): открой раздел, нажми в браузере «Показать код страницы»
(View Source, не DevTools) — в исходном HTML должны быть карточки товаров
со ссылками /product/... (а не пустой grid).

ОСТАЁТСЯ (по желанию): страница брендов, страница избранного/сравнения,
подключение домена servis-com.kz к Render (канонические ссылки и sitemap
уже настроены на него).
