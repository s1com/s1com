СЕРВИС.COM — ВСЕ ИЗМЕНЕНИЯ (единый архив, ВЕБ-сервис)
====================================================
Один деплой на веб-сервис. Импорт/крон запускать НЕ нужно.
Заливать ВСЕ файлы разом — так админка и сервер согласованы (без «мёртвых кнопок»).

ФАЙЛЫ В АРХИВЕ (8):
  server.js            — FAQ-эндпоинты, статусы заявок + сумма, статистика
                         продаж/конверсии, 301 /index.html→/, honeypot и
                         обработка ошибок в заявке, SSR-товары в разделах.
  db.js                — таблицы suppliers/offers/faq (+сиды) и поле суммы заявок.
  admin/index.html     — вкладки «Поставщики» и «FAQ»; в заявках статусы
                         Новая/В работе/Заказ/Отказ + поле суммы; в Дашборде
                         блок «Сделки и конверсия».
  lib/product-page.js  — характеристики без обрезки; кнопка WhatsApp с
                         названием+артикулом + строка доверия.
  public/index.html    — динамический FAQ (из админки) + canonical на «/».
  public/videonablyudenie.html, setevoe.html, skud.html, pozharnaya.html
                       — динамический FAQ (со статическим SEO-fallback).

НЕ ВКЛЮЧЕНО (намеренно):
  • app.js, style.css — каталог возвращён к твоему привычному виду
    (мегаменю/верхние фильтры убраны), эти файлы не менялись.
  • scripts/alstyle-import.js — это крон, уже задеплоен и работает.

ДЕПЛОЙ:
 1) GitHub → Add file → Upload files → перетащить в КОРЕНЬ:
       server.js, db.js
    и папки целиком (со структурой):
       admin/   (index.html)
       lib/     (product-page.js)
       public/  (5 html)
    → Commit в main.
 2) Render → ВЕБ-сервис → Manual Deploy → «Clear build cache & deploy» →
    дождаться статуса Live.
 3) Открыть сайт и /admin с Cmd+Shift+R.
