- **`GET /api/admin/duplicates`** (auth): находит группы «один бренд + нормализованная модель в РАЗНЫХ `source`» (Al-Style ⇄ Complex/Excel). Ограничивается брендами, что в >1 источнике (эффективно на 57k). Возвращает группы с `id/sku/source/visible/price/stock`.
- **Админка** (🏭 Поставщики → «🔁 Найти дубли между поставщиками»): показывает группы, у каждого товара кнопка «Скрыть» → существующий bulk-hide (`{ids,action:'hide'}`). Оставляешь нужный источник, лишний скрываешь. **Без авто-мёрджа/удаления — безопасно.**
- Проверено (supplier-integration 15/15): детект дубля в 2 источниках, id для скрытия, 401 без токена, после hide лишний visible=0.
- **НЕ делал (крупно/рискованно):** авто-склейка по бренд+MPN/EAN в единый оффер, очередь конфликтов `match_queue`, `price_rules` UI — отдельная большая фича, нужна при массовом пересечении каталогов. Детектор+ручное скрытие покрывает текущую потребность.
