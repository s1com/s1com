ПОЛНОЕ ДЕРЕВО КАТЕГОРИЙ (как у Al-Style)
=========================================

Что сделано: подкатегории видеонаблюдения (и всех разделов) теперь — полное
многоуровневое дерево Al-Style. Например:
   Видеонаблюдение → IP видеокамеры → Купольные IP → 2/4/5/8 МП
                                    → Цилиндрические IP → 2/4/5/8 МП
                                    → PT и PTZ → 2/4/5/8 МП
   Видеонаблюдение → Wi-Fi видеокамеры → … → по мегапикселям
   Видеонаблюдение → IP видеорегистраторы → 4/8/16/32-канальные

Категории хранятся по ID Al-Style (а не по имени) — поэтому одинаковые
«2 мегапиксельные» под Купольными и Цилиндрическими больше НЕ конфликтуют.
Каждый товар несёт путь категорий, по нему и фильтруется дерево слева.

ФАЙЛЫ К ЗАМЕНЕ (перетащите в GitHub с сохранением папок):
  db.js
  server.js
  scripts/alstyle-import.js
  scripts/seed.js
  public/js/app.js
  public/css/style.css
  admin/index.html

ДЕПЛОЙ — по шагам:
  1) GitHub → Add file → Upload files → перетащите СОДЕРЖИМОЕ этой папки
     (db.js, server.js, папки scripts/ public/ admin/ — НЕ саму обёртку).
  2) Commit changes (в main).
  3) Render (servis-catalog) → Manual Deploy → Deploy latest commit. Дождитесь Live.
  4) ⚠ ОБЯЗАТЕЛЬНО запустите импорт: Render → ваш Cron → Trigger Run.
     Дерево и пути категорий заполняются ТОЛЬКО при импорте.

ВАЖНО (прочитайте перед деплоем):
  • После деплоя и ДО первого импорта раздел категорий будет пустым — это норма.
    Запустите Cron (шаг 4) — дерево соберётся само из Al-Style.
  • Старая таблица категорий пересоздаётся под новую схему. Ручные настройки
    видимости категорий сбросятся (заполнятся заново при импорте, всё видимо).
  • Ручное добавление категорий в админке отключено — дерево формируется из
    импорта автоматически. Чтобы скрыть лишние ветки на сайте — снимайте галочку
    «Показ» в админке → 🗂 Категории (скрытие СОХРАНЯЕТСЯ между импортами,
    скрытие родителя прячет и все его подкатегории).

ПРОВЕРКА после импорта:
  • На сайте слева в «Видеонаблюдение» появляется дерево с раскрытием:
    клик по «IP видеокамеры» → показываются Купольные/Цилиндрические/PTZ;
    клик по «Купольные» → показываются 2/4/5/8 МП; клик по уровню фильтрует товары.
  • Что код залит: откройте /js/app.js и найдите (Ctrl+F) слово recomputePool —
    если есть, новая версия на месте.
  • Админка → 🗂 Категории: дерево с отступами по уровням, галочки видимости,
    счётчик товаров в каждой ветке (с учётом подветок).
